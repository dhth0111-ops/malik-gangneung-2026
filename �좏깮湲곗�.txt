기준본 선택: SundayPMMuseum_v1.0_관리형.zip

선택 이유
- 세 후보 중 index.html 규모와 기능 범위가 가장 큼
- 홈, 시즌, 생활권, 맛집, 관리 탭 포함
- 장소/시즌/생활권/맛집 추가·수정·삭제 포함
- 방문 날짜 여러 번 기록, 수정, 삭제, 방문 횟수와 최근 방문일 표시 포함
- 대표사진과 사진명소 이미지를 IndexedDB에 저장
- 사진까지 포함한 JSON 백업/복원 포함
- PWA 구성(manifest, service worker, icon) 포함

검증
- JavaScript 문법 검사 통과
- ZIP 구성 파일 확인 완료
