Malik Trip V7.1 Stable

강릉 여행이 정상 표시되던 V7.0 다중여행 버전을 기준으로 제작했습니다.

추가 기능
- 여행명에 따른 배너 문구 자동 변경
- 일정 아이콘에 여행 시작일 표시
- 더보기에서 여행명/시작일/종료일 수정
- 여행별 대표사진 변경 및 기본사진 복원

안전 원칙
- V7.0의 여행 목록(listenTrips) 로직은 변경하지 않았습니다.
- Firebase 설정과 sharedTrips 데이터 구조를 변경하지 않았습니다.
- 강릉 문서를 새로 만들거나 덮어쓰는 별도 복구 코드를 추가하지 않았습니다.
