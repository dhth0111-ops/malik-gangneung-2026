Malik Trip V7.1 Safe Rebuild

기준
- V7.0 다중여행 원본에서 재구성
- 사과머리 / 꼬꼬마 / 아기천사 아이콘 포함
- 기존 sharedTrips 문서 구조 유지
- 여행 목록 로직 유지

- 저장 전 최근 20개 상태를 기기 자동백업
- 현재 여행 JSON 백업 / 가져오기
- 최근 자동백업 복원

- 강릉 기본 데이터를 자동으로 덮어쓰지 않음
- 서비스워커 캐시 버전 분리
