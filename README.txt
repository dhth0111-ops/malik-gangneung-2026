요리왕 꼬꼬마 V2.2.1

핵심 수정
- 사용자가 수정한 레시피 내용, 사진, 즐겨찾기, 카테고리를 덮어쓰지 않음
- 기본 레시피가 중복된 경우 사용자 수정 흔적이 많은 항목을 우선 보존하고 중복만 제거
- 동일하거나 비어 있는 레시피 ID를 안전하게 재정리
- 기존 IndexedDB + localStorage 이중 저장 및 자동 백업 유지
- 없는 기본 레시피만 최초 1회 추가
- 레시피 추가, 수정, 삭제 기능 유지
- 대표사진 중심 UI, 카테고리, 즐겨찾기 유지

중요
- 코로나김밥은 기본 레시피에 강제로 넣지 않음
- 사용자가 직접 만든 코로나김밥은 그대로 보존됨
- 업데이트 전 '백업 저장'으로 JSON 백업을 한 번 만들어 두는 것을 권장
